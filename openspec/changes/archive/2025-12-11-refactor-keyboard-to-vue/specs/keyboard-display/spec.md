## MODIFIED Requirements

### Requirement: 虚拟键盘显示
系统 SHALL 显示虚拟键盘，使用纯 HTML/CSS 实现（非 SVG/Canvas），布局模拟真实 QWERTY 键盘。

#### Scenario: 显示键盘布局
- **WHEN** 用户进入打字练习页面
- **THEN** 显示完整的键盘布局
- **AND** 使用 CSS Grid/Flexbox 实现行布局
- **AND** 每行有适当的偏移以模拟真实键盘

#### Scenario: 手指颜色区分
- **WHEN** 键盘显示时
- **THEN** 每个按键根据对应手指显示不同的边框颜色
- **AND** 颜色与底部手部指示器一致

#### Scenario: 高亮目标键
- **WHEN** 有目标字符需要输入
- **THEN** 对应按键背景填充为手指颜色
- **AND** 显示发光/脉冲动画效果
- **AND** 按键文字变为白色以保持可读性

#### Scenario: 按键反馈
- **WHEN** 用户按下键盘上的键
- **THEN** 对应的虚拟按键显示按下动画
- **AND** 根据正确/错误显示不同的颜色反馈

#### Scenario: 显示手部提示
- **WHEN** 键盘显示时
- **THEN** 在底部显示左右手的指示区域
- **AND** 每个手指用彩色圆点表示
- **AND** 圆点尺寸足够大便于识别

#### Scenario: 目标手指高亮
- **WHEN** 有目标字符需要输入
- **THEN** 对应的手指圆点完全高亮（不透明度 100%）
- **AND** 显示明显的脉冲/闪烁动画
- **AND** 其他手指保持较低不透明度
